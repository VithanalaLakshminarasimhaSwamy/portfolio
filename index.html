Academic Projects

Signature Verification System using CNN

2025

This project uses a Convolutional Neural Network (CNN) to automatically verify handwritten signatures by comparing them with genuine reference samples. The model is trained on a dataset of real and forged signatures, learning to identify subtle differences in writing patterns, stroke pressure, and shape. It helps in detecting fraudulent signatures with high accuracy and can be used in banking, document verification, and identity authentication systems.

2025
